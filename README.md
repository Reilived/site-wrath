# Wrath PvP Website Hosting Guide

To host this site 24/7 for free:

### 1. Upload to GitHub
1. Create a repository on [GitHub](https://github.com) called `wrath-site`.
2. Upload all files from this `Site` folder (except `node_modules`).

Recommended repo name: `site-wrath`

### 2. Host on Render.com
1. Go to [Render.com](https://render.com).
2. Click **New +** -> **Web Service**.
3. Connect your GitHub account and pick `wrath-site`.
4. Use these settings:
   - **Environment**: `Node`
   - **Build Command**: `npm install`
   - **Start Command**: `node server.js`
5. Click **Create Web Service**.

### 3. Open Port on Minecraft VPS
The website on Render needs to talk to your Minecraft server on port **19000**. 
Ensure port **19000** is open in your VPS firewall.
